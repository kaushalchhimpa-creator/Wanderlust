# 🌍 Wanderlust - Full-Stack Travel Marketplace

Wanderlust is a feature-rich, full-stack web application designed to simulate an Airbnb-like accommodation exploration and booking platform.

## 🛠️ Key Features
* **User Lifecycle (Auth):** Secure Sign-up, Log-in, and Session tracking.
* **Data Privacy & Authorization:** Robust middleware checks—users can only edit/delete their own listings or personal reviews.
* **Full CRUD Operations:** Seamless management of travel destinations, pricing, and dynamic review states.
* **Location Analytics:** Integrated map interfaces rendering visual landmarks for properties.
